E-commerce Store Development Prompt

Overview

Create a modern, fully-functional e-commerce store that mimics the core features and user experience of Shopify. The store should be professional, responsive, and include all essential e-commerce functionality.

Core Features Required

Frontend/Customer-Facing Features

  • Product Catalog: Grid and list views with filtering and sorting options
  • Product Pages: Detailed product information, image galleries, variants (size, color, etc.)
  • Shopping Cart: Add/remove items, quantity adjustment, persistent cart
  • Checkout Process: Multi-step checkout with guest and registered user options
  • User Accounts: Registration, login, order history, wishlist
  • Search Functionality: Product search with autocomplete and filters
  • Navigation: Intuitive menu structure with categories and subcategories
  • Responsive Design: Mobile-first approach, works seamlessly across all devices

Essential Pages

  • Homepage with featured products and promotions
  • Category/collection pages
  • Individual product pages
  • Shopping cart and checkout pages
  • User account dashboard
  • About Us, Contact, FAQ, Privacy Policy pages
  • Order confirmation and tracking pages

Payment & Shipping

  • Multiple payment options integration ready
  • Shipping calculator and options
  • Tax calculation support
  • Discount codes and promotions system

Design Requirements

  • Modern, clean aesthetic similar to Shopify themes
  • Professional typography and color scheme
  • High-quality product image display
  • Intuitive user interface with clear call-to-action buttons
  • Loading states and smooth transitions
  • Accessibility compliance (WCAG guidelines)

Technical Specifications

  • Fast loading times and optimized performance
  • SEO-friendly structure
  • Cross-browser compatibility
  • Secure data handling
  • Form validation and error handling
  • Local storage for cart persistence

Nice-to-Have Features

  • Product reviews and ratings
  • Recently viewed products
  • Related/recommended products
  • Email newsletter signup
  • Social media integration
  • Live chat support widget
  • Inventory level indicators
  • Product comparison feature

User Experience Goals

  • Intuitive navigation that makes finding products effortless
  • Streamlined checkout process to minimize cart abandonment
  • Professional appearance that builds trust and credibility
  • Fast, responsive performance across all devices
  • Clear product information and pricing
  • Easy account management for returning customers

Success Metrics

  • Clean, professional design that rivals top Shopify stores
  • All core e-commerce functionality working seamlessly
  • Responsive design that works perfectly on mobile and desktop
  • Intuitive user flow from product discovery to purchase completion
  • Proper form validation and error handling throughout
